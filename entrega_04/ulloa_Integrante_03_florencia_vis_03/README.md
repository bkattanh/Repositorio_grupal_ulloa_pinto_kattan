# README -- Florencia Ulloa

## 1. Proceso y decisiones de visualización

**Objetivo:**\
Comunicar la brecha de participación de atletas únicos entre **Salto
Largo** y **Salto con Garrocha** por categoría, estableciendo el
problema como proyección final para la webstory grupal.

### Decisiones de Diseño

-   **Visualización Atómica:**\
    Se seleccionó un **gráfico de barras agrupadas** para permitir la
    comparación directa del número de atletas en ambas disciplinas
    dentro de cada categoría. Esta disposición resalta visualmente la
    magnitud de la brecha de participación.

-   **Codificación de Color (Azul/Blanco):**\
    Se utilizó **Azul Primario (#0D47A1)** para *Salto Largo* (alta
    participación) y **Azul Secundario (#42A5F5)** para *Salto con
    Garrocha* (baja participación).

-   **Tipografía:**\
    Se empleó **Cooper Hewitt** 

-   **Rol Narrativo:**\
    Esta visualización funciona como el **Capítulo III** de la webstory,
    validando la hipótesis de que las diferencias de rendimiento
    observadas posteriormente deben interpretarse como consecuencia de
    un problema de acceso y masividad en la base de atletas.

## 2. Descripción de la lectura visual

**Lectura visual:**\
El afiche titulado **"LA BASE SE CAE"** establece un tono crítico y
directo.
La visualización presenta las barras de *Salto Largo* (azul primario)
claramente más altas que las de *Salto con Garrocha* (azul secundario),
destacando una brecha especialmente notoria en las **categorías
femeninas y escolares**.\
Esta diferencia visual actúa como evidencia del problema estructural.

**Potenciación del mensaje:**\
La prosa complementa el gráfico interpretando los datos, no solo
describiéndolos.\
Se argumenta que la disparidad responde a **factores estructurales**
como el acceso a equipamiento o los cupos disponibles, transformando un
dato cuantitativo en un **insight periodístico**.

**Transparencia de datos:**\
El uso de **atletas únicos** como métrica principal refuerza la
rigurosidad y credibilidad del análisis, otorgando sustento empírico al
discurso visual.

**Coherencia visual:**\
La elección de una paleta en tonos azules, en lugar de colores de
"alerta" (como el naranja previo), busca proyectar **confianza, método y
profesionalismo**.
